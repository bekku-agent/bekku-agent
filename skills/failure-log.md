# Failure Log

Every failure is a lesson. This file compounds learnings so the same mistake never happens twice.

## Format
```
### [Date] — [What Failed]
**What happened:** Brief description
**Root cause:** Why it failed
**Fix:** What solved it
**Rule:** The permanent rule to prevent recurrence
```
